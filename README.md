# Gran's House Tattoos

A bespoke, cosy, and "fun" visual identity for an Edinburgh-based tattoo studio. This project reimagines the traditional, high-end heritage aesthetic with a lighter, more welcoming "Gran's house" feel — complete with tea, shortbread, and quirky side embellishments.

## ✨ Project Highlights

- **Breaking the Grid**: Incorporates massive, quirky side elements (teapots, Irn-Bru cans, floor lamps, and more) that "break" the central layout to create a layered, organic aesthetic.
- **Scroll Parallax**: Key side elements feature subtle vertical parallax effects, adding depth and life to the scroll experience.
- **Heritage Styling**: Uses custom gingham and tartan patterns to pay homage to Scottish heritage without feeling formal or outdated.
- **Responsive Design**: Carefully optimized for all screen sizes, with side elements that proportionally scale or tuck away to ensure a clean mobile experience.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom vanilla CSS for layouts, complex gradients (gingham patterns), and animations.
- **JavaScript**: Minimal, performance-oriented script for mobile navigation, scroll-reveal observers, and parallax logic.

## 📁 Key File Structure

- `/assets/images/`: Contains all branding assets and the large "breaking the grid" PNGs/WebPs.
- `index.html`: The main page content and structure.
- `style.css`: The complete design system, layout, and embellishment styles.
- `script.js`: Interactive functionality including header shrinking and parallax scrolling.

## 🍵 Background

The studio name, Gran's House, is built on the warmth of home. The website's design reflects this by using a soft palette of sky blue, warm red, and cream, moving away from the typical dark aesthetic of traditional tattoo studios.
